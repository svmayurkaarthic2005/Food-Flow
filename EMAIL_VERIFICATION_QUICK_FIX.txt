🚨 TROUBLESHOOTING

Email Not Received?
   1. Check spam/junk folder
   2. Verify backend is running (http://localhost:8000/health)
   3. Check backend console for "✅ Email sent to..."
   4. Try resend: http://localhost:3000/auth/resend-verification

Still Getting "Invalid verification link"?
   1. Make sure you clicked the LATEST email
   2. Check token hasn't expired (24h)
   3. Verify both backend AND frontend are running
   4. Check browser console for errors

Backend Connection Failed?
   1. Start backend: cd backend && python -m uvicorn main:app --reload
   2. Check port 8000 is not in use
   3. Verify NEXT_PUBLIC_API_URL in frontend/.env.local
