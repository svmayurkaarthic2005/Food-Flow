📧 HOW EMAIL VERIFICATION WORKS

   1. User signs up at /signup
   2. Frontend creates account in database
   3. Frontend calls backend email service
   4. Backend sends email via Brevo SMTP
   5. User receives professional email
   6. User clicks verification link
   7. Account verified! ✅

   If backend is not running:
   → Frontend logs verification URL to console
   → You can still test the flow
   → Email just won't be sent
